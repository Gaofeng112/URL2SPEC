3、课题名称：智能接口测试智能体（优先级 B）
需求背景：
目前公司部分项目没有接口文档，很难开展接口测试。需要先采集现有接口的数据，人工判断接口功能再补全接口测试脚本。
希望能智能爬取与分析接口的内容，生成接口文档，然后再对接口进行测试。
预期交付成果：
智能体：给定一个页面url，就能够智能爬取并分析当前页面所有接口的功能与参数，给出高可用的接口文档，生成接口测试脚本，输出测试报告。

流程：
用户输入 URL=https://www.yaozh.com/ 这里以药智网为例
    ↓
[1] 页面访问模块
    - Playwright 打开页面
    - 等待页面加载
    - 捕获 XHR / Fetch 请求

    ↓
[2] 接口采集模块
    - method
    - url
    - query 参数
    - request body
    - response body
    - status code
    - headers

    ↓
[3] 数据预处理模块
    - 去重
    - 过滤静态资源
    - 敏感字段脱敏
    - 响应体截断
    - JSON Schema 推断

    ↓
[4] LLM 分析模块 openai的创建llm
    - 推测接口名称
    - 分析接口功能
    - 解释请求参数
    - 解释响应字段
    - 生成注意事项
    - 生成接口文档结构

    ↓
[5] 文档生成模块
    - Markdown 文档
    - OpenAPI
